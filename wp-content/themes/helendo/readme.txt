Helendo - Minimalist AJAX WooCommerce WordPress Theme
------------------------------

Helendo is a a clean & minimal AJAX WooCommerce WordPress Theme for shopping online stores. This theme is suited for a lot of e-commerce website such as fashion store, furniture store, decoration store, etc.
